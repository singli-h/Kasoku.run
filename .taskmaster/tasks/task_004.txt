## Action File Organization

### Clear Separation of Concerns

**Periodization** ([training-plan-actions.ts](apps/web/actions/training/training-plan-actions.ts)):
- Pages: `/plans`, `/plans/[id]`
- Purpose: Long-term training plan structure

**Session Planning** ([session-plan-actions.ts](apps/web/actions/training/session-plan-actions.ts)):
- Pages: MesoWizard workflow
- Purpose: Create multiple sessions at once

**Session Editing** ([session-planner-actions.ts](apps/web/actions/training/session-planner-actions.ts)):
- Pages: `/plans/[id]/session/[sessionId]`
- Purpose: Edit individual session with full exercise/set management
- **Key Action**: `saveSessionWithExercisesAction` - Atomic save of session + exercises + sets

**Exercise Library** ([exercise-actions.ts](apps/web/actions/training/exercise-actions.ts)):
- Pages: `/library`, Session Planner (exercise picker)
- Purpose: Browse, create, edit exercises

**Races** ([race-actions.ts](apps/web/actions/training/race-actions.ts)):
- Pages: `/plans/[id]` (race markers on timeline)
- Purpose: Track competitions for periodization

```typescript
// Exercise Library
import { getExercisesAction, getExerciseTypesAction } from '@/actions/training'

// Race Management
import { getRacesByMacrocycleAction, createRaceAction } from '@/actions/training'

// Validation
import { RaceSchema, SessionSchema } from '@/lib/validation/training-schemas'
```

### Common Operations
```typescript
// Save session with all exercises and sets
const result = await saveSessionWithExercisesAction(
  sessionId,
  { name, description, date },
  exercises
)

// Load exercises for library
const exercises = await getExercisesAction({ search: 'squat' })

// Create race
const race = await createRaceAction({
  name: 'Boston Marathon',
  type: 'Marathon',
  date: '2025-04-21',
  macrocycle_id: 1
})
```
