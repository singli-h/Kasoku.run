# Task 3: Standardize Layout and Fix Header Duplication

## Description
Ensure consistent ProtectedLayout usage without header duplication and implement proper page containers and spacing.

## Details
1. Audit all pages to identify inconsistent layout usage and header duplication.
2. Create a standardized ProtectedLayout component that includes the header.
3. Implement a flexible container system using CSS Grid or Flexbox for consistent spacing.
4. Add error boundaries at the page level using react-error-boundary v4.0.4.
5. Ensure responsive design using CSS media queries and relative units.

Example implementation:
```typescript
// components/ProtectedLayout.tsx
import { ErrorBoundary } from 'react-error-boundary';
import Header from './Header';

const ProtectedLayout: React.FC<{ children: React.ReactNode }> = ({ children }) => (
  <ErrorBoundary FallbackComponent={ErrorFallback}>
    <div className="layout-container">
      <Header />
      <main className="content-container">{children}</main>
    </div>
  </ErrorBoundary>
);

// styles/layout.css
.layout-container {
  display: grid;
  grid-template-rows: auto 1fr;
  min-height: 100vh;
}

.content-container {
  padding: 2rem;
  max-width: 1200px;
  margin: 0 auto;
  width: 100%;
}

@media (max-width: 768px) {
  .content-container {
    padding: 1rem;
  }
}
```

Usage in pages:
```typescript
// pages/sessions.tsx
import ProtectedLayout from '../components/ProtectedLayout';

const SessionsPage: NextPage = () => (
  <ProtectedLayout>
    <GroupSessionDashboard />
  </ProtectedLayout>
);
```

## Test Strategy
1. Implement visual regression tests using tools like Percy or Chromatic to ensure consistent layout across pages.
2. Write unit tests for the ProtectedLayout component using React Testing Library.
3. Use Cypress for end-to-end tests to verify proper layout rendering and responsiveness.
4. Test error boundary functionality by intentionally throwing errors in child components.
5. Verify accessibility using automated tools like axe-core and manual keyboard navigation tests.
